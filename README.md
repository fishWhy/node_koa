# coderhub
A coderhub developed using node koa

该基于node koa的后端项目，源自codeWhy。仰慕该大神对项目结构的划分，方便日后在写node koa时能够参考而创立的。

目录结构的介绍可参见： 项目目录简介.png

#### 项目功能包括：

1.用户管理系统

2.内容管理系统

3.内容评论管理

4.内容标签管理

5.文件管理系统。

具体可见ppt



#### 该项目可参考点：

1. 目录结构划分(即项目的整体结构)

2. 应用配置信息写到环境变量

3. koa创建和启动服务器

4. 用户注册、登录、JWT实现token验证

5. 与mysql数据库交互

6. 错误统一处理

7. 图片上传与获取
